ManuscriptMatch

A platform connecting writers with beta readers and literary agents

Year 2025
Role Solo Developer
Next.jsPostgreSQLTypeScript
Demonstration

The Problem

Writers struggle to find beta readers and connect with literary agents.
The query process is fragmented and opaque.

The Solution

A full-stack platform with features for every stakeholder:

For Agencies: Agency pages, agent management dashboards, invite system

For Agents: Customizable query letter wizards, wishlist profiles, query management dashboard

For Writers: Agent matching system, manuscript upload, query tracking with status updates

For Beta Readers: Apply to read manuscripts, in-app annotation tools, reputation/rating system

Smart genre matching and analytics tie everything together.
